# PalOps 项目协作约定

本文件只维护本仓库特有且长期稳定的协作约定。通用沟通、安全、范围和交付规则继续遵循上层约定。

## 隐私与本机配置

- 不在已跟踪文件、提交消息或测试夹具中记录真实玩家 UID、昵称、玩家存档名、世界 ID、容器 GUID、SSH 主机名或宿主机绝对路径。
- 本机专用连接与角色设置保存在已忽略的 `.paledit-data/settings.json`；批量箱子目标保存在已忽略的 `.paledit-data/box-targets.json`。
- 文档和测试只使用 `<WorldId>`、`palworld-server`、`/srv/palworld` 等通用示例或明确生成的合成数据。

## 身份解析

- 分析“我的角色”“我的存档”“我的装备”或“我的帕鲁”时，以当前本地设置和当前存档中的身份映射为准。
- 不根据在线玩家、最高等级、昵称相似或最近活跃情况猜测用户身份。

## 存档与世界定位

- 本仓库本地存档根目录为 `Save/`。
- 世界目录通过 `Save/SaveGames/0/*/Level.sav` 动态发现，不把某个世界 ID 视为仓库常量。
- 远端连接信息以 `.paledit-data/settings.json` 为真源；需要判断“目前”或“最新”状态时，应先核对远端与本地存档时间或显式刷新，不能默认本地副本仍是最新版本。
