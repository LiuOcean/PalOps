# PalEdit 项目协作约定

本文件补充本仓库特有的稳定事实与身份映射；通用协作、安全和交付规则继续遵循上层约定。

## 用户角色身份

- 用户本人在当前 Palworld 世界中的角色 UID 固定为 `00000000-0000-0000-0000-000000000000`。
- 当前存档中的角色昵称为 `测试角色`。
- 对应玩家存档文件为 `Players/00000000000000000000000000000000.sav`。
- 分析“我的角色”“我的存档”“我的装备”或“我的帕鲁”时，只以该 UID 归属的数据为准；不要根据在线玩家、最高等级、昵称相似或最近活跃情况猜测用户身份。
- 昵称、等级、装备、背包、帕鲁与其他成长数据可能变化，执行任务时必须从当前存档重新读取；只有角色 UID 和玩家文件映射可作为稳定定位信息。

## 当前世界定位

- 本仓库本地存档根目录为 `Save/`。
- 当前世界 ID 为 `00000000000000000000000000000000`。
- 当前世界目录为 `Save/SaveGames/0/00000000000000000000000000000000/`。
- 远端存档真源位于 SSH 主机 `palworld-server`；需要判断“目前”或“最新”状态时，应先核对远端与本地存档时间或显式刷新，不能默认本地副本仍是最新版本。
